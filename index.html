Retail Sales Data Analysis

Cleaned and prepared a retail sales dataset in PostgreSQL by removing null value records and executing advanced SQL queries. Connected the PostgreSQL database to Power BI Desktop to create visualizations and conduct further sales trend analysis.

Amazon sales data analysis

I worked with a Kaggle dataset of 120K+ rows, tackling missing values and inconsistencies using Python (Pandas). Performed extensive data cleaning and transformation to prepare it for analysis. Used SQL to filter, aggregate, and reshape the data for deeper insights. Built an interactive Power BI dashboard showing category-wise sales, monthly trends, city performance, and promo impact. Published the dashboard via Power BI Service (Microsoft Fabric) for seamless sharing and collaboration.
